go-sieve
====================

Sieve email filtering language ([RFC 5228])
implementation in Go.

## Supported extensions

- envelope ([RFC 5228])
- fileinto ([RFC 5228])
- imap4flags ([RFC 5232])

## Example

See ./cmd/sieve-run.
